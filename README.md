## To Run Project
### Install libraries
``` bash
cd PythonServer
pip install -r requirements.txt
```

### Run Python Backend Server: 
In the terminal, to run the python code
``` bash
# for python
python app.py
# for python3
python3 your_script.py
```

### Running Next.js front end:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
